## 🎓 Academic Documentation

The original graduation project documentation is included in this repository.

The academic scope of the university project focused only on the implementation of the humanoid auto-rigging tool later known as **IziHuman**.

Additional tools and the broader suite concept were developed after the academic project.

Current suite scope:

- IziHuman
- IziCurve
- IziRename
- Planned modules

---

## 🌎 Language Notice

The graduation project document is currently available **only in Spanish**.

The document was produced for academic delivery in **Costa Rica**, therefore the university requirements required the project to be written in Spanish.

At the moment there is **no official English translation** available.
