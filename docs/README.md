## 🎓 Graduation Project

The original graduation project documentation is included in this repository.

The academic scope focused only on the implementation of the humanoid auto-rigging system later known as **IziHuman**.

Additional tools and the broader suite concept were developed after the academic delivery.

Current suite scope:

- IziHuman
- IziCurve
- IziRename
- Planned modules
