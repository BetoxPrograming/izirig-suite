# User Feedback

This document summarizes early user feedback collected during the IziRig development process.



---

## Feedback Summary

The participant identified several important expectations for an auto-rigging tool:

- Facial rig support
- IK/FK global and local switching
- Stable knee and elbow behavior
- More limb options
- Support beyond humanoid characters
- Lightweight performance

---

## Design Impact

This feedback supported the need to plan IziRig as a modular toolset instead of limiting it only to humanoid characters.

It also reinforced the importance of keeping the tool lightweight, flexible and expandable.
